# Pave Over MoCo

An independent advocacy site for road capacity, congestion relief, and accountable
transportation spending in Montgomery County, Maryland.

Live at [paveovermoco.com](https://paveovermoco.com) via GitHub Pages.

## Structure

- `index.html` — homepage: platform, data, county spotlight, pledge
- `road-expansion.html` — Policy Brief 01: the case for road capacity
- `bike-lanes.html` — Policy Brief 02: bike lanes, demand before concrete
- `bus-lanes.html` — Policy Brief 03: transit spending, results first
- `privacy.html` — privacy policy
- `404.html` — not-found page
- `styles.css` / `script.js` — shared styles and behavior (no frameworks, no build step)

All statistics on the site are attributed inline; source links appear in each
brief's sidebar and in the site footer.
